Node.js can be easily installed from the installation setup provided at the <a href="https://nodejs.org">official Node.js website</a>. Download the setup provided, and continue with the instructions in it.
<br/>
<a href="http://zocada.com/wp-content/uploads/2018/01/nodejsorg_website_windows_download.png"><img src="http://zocada.com/wp-content/uploads/2018/01/nodejsorg_website_windows_download.png" alt="nodejs.org website" width="879" height="646" class="alignnone size-full wp-image-121" /></a>

Once the installation completes. Open the command prompt by typing <code>cmd</code> in the Run window (win_key + R). Type the <code>node -v</code> and you should be able to see the node.js version installed on your windows.
